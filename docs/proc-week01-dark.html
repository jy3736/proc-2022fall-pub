Python 簡介

  • Python 是1991年公開發佈的物件導向腳本語言(Object-oriented Scripting Language)，由阿姆斯特丹國家數學和計算機科學研究所的 Guido van Rossum 開發。
  • Python 是開源(Open Source)的、免費的，並且擁有廣大的開源社群的支持。
  • 不論是對入門或是專業的程式設計師， Python 是一個容易上手入門且可讀性高的語言。
  • Python 擁有類別廣泛且強大的標準函式庫及第三方開源庫，能有效提升程式設計師的產出，且以最少的程式碼完成複雜的專案。
  • Python 提供多種近代的程式設計範式(programming paradigm)，例如 Procedural、Functrional、Object-oriented、及 Reflective 等。
  • 不論是在資料科學(Data Science)、AI深度學習(Deep Learning)、若是前後端網頁程式設計(Web Programming)，Python 都是最受歡迎的程式語言之一，因此網路上有非常多的免費且開源的相關應用程式。
  • Python 不但可以快速建構小型專案，同時也擁有開發大型專案的可朔性，因此很多知名的IT公司都有以Python開發的商用專案，如 Google、Facebook、Dropbox、 YouTube、 Reddit、 Instagram、Quora、等等。
  • Python 簡化了平行程式的開發——使用 asyncio 和 async/await，程式設計師可以輕鬆編寫單線程平行程式。
  • 還有很多使得 Python 成為最受歡迎語言的原因，持續發生中....
