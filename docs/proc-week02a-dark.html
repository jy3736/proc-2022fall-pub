課堂作業

課堂作業 1：Serial-port Controlled LED 繳交期限：繳交期限：9/19

  • Receive control command N from TX/RX( serial port):
    • N = 1, set the LED blinking cycle to 200ms
    • N = 2, set the LED blinking cycle to 1s
    • N = 3, set the LED blinking cycle to 2s
    • Others, print "Unknown command!" to the serial port

void setup()
{
  pinMode(13, OUTPUT);
  Serial.begin(9600);
}

int dly = 100;
int ledst = LOW;

void loop()
{
  digitalWrite(13,ledst);
  if (Serial.available()) {
    int cmd = Serial.parseInt();
    Serial.print("Received cmd = ");
    Serial.println(cmd);
    switch(cmd) {
      case 1 : dly = 100; break;
      case 2 : dly = 500; break;
      case 3 : dly = 1000; break;
      default: Serial.println("Unknown command!"); break;
    }
  }
  ledst = ~ledst;
  delay(dly);
}

課堂作業 2：Button/Serial Control Traffic Light 繳交期限：9/19

  • Receive control command N from TX/RX( serial port):
    • N = 0 : all LEDs are off
    • N = 1 : turn on the red LED
    • N = 2 : red LED flashes with a period of 500ms
    • N = 3 : turn on the yellow LED
    • N = 4 : yellow LED flashes with a period of 500ms
